# Commands

## Dangerously Skip Permissions Mode

```bash
claude --dsp
```

**Alias für:** `claude --dangerously-skip-permissions`

**Funktion:** Startet Claude Code im bypass permissions mode - überspringt alle Tool-Berechtigungsabfragen und erlaubt Claude, alle Tools ohne Nachfrage zu verwenden.

**Warnung:** Dieser Modus sollte nur in vertrauenswürdigen Umgebungen verwendet werden, da Claude uneingeschränkten Zugriff auf alle System-Tools erhält.
